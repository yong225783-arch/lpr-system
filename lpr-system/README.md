# 🚗 停車場管理系統

一個專業的停車場管理系統，支援車牌辨識、計費管理、車位管理，透過網頁進行管理操作。

## 功能

### 核心功能
- ✅ **車牌辨識** - 即時影像辨識（OpenCV + EasyOCR）
- ✅ **自動開門** - 白名單車牌自動開門
- ✅ **網頁管理** - 新增/編輯/刪除車主資料
- ✅ **開門紀錄** - 完整時間、車牌、車主、狀態記錄
- ✅ **手動開門** - 網頁按鈕直接開門
- ✅ **黑白名單** - 支援黑名單封鎖
- ✅ **照片存證** - 開門時自動截圖

### 🚀 新增功能
- ✅ **車位管理** - 可視化車位狀態（可用/已占用/預訂中/停用）
- ✅ **停車計費** - 靈活的收費規則設定
- ✅ **自動化計費** - 進場/離場自動計算費用
- ✅ **帳單管理** - 完整的帳單記錄和統計
- ✅ **手動進場/離場** - 支援手動操作

## 快速開始

### 1. 安裝 Python

到 https://www.python.org/downloads/ 下載 Python 3.10+，安裝時勾選 **Add Python to PATH**。

### 2. 安裝依賴

```bash
cd lpr-system
pip install -r requirements.txt
```

### 3. 啟動系統

```bash
# Linux/macOS
chmod +x start.sh
./start.sh

# Windows
python main.py
```

### 4. 開啟網頁

```
http://localhost:5000
```

- 帳號：`admin`
- 密碼：`admin123`

## 環境變量

| 變量 | 預設值 | 說明 |
|------|--------|------|
| `CAMERA_SOURCE` | `0` | 攝影機來源（0=USB webcam, RTSP URL=IP Cam） |
| `RELAY_PORT` | - | USB 繼電器 Port（如 COM3） |
| `SIMULATE_RELAY` | `1` | 設為 1 啟用模擬模式 |

## 收費範例

假設收費規則：
- 免費：15 分鐘
- 基本費：NT$ 0
- 每小時：NT$ 30
- 每日上限：NT$ 500

| 停車時長 | 費用計算 | 費用 |
|----------|----------|------|
| 10 分鐘 | 在免費範圍內 | NT$ 0 |
| 30 分鐘 | (30-15)/60 小時 = 1小時 | NT$ 30 |
| 2 小時 | (120-15)/60 = 1.75 → 2小時 | NT$ 60 |
| 24 小時 | 費用超過上限 | NT$ 500 |

## 故障排除

**網頁無法開啟**
- 確認 Python 程式有正常執行
- 確認 port 5000 未被其他程式佔用

**繼電器無反應**
- 確認 `SIMULATE_RELAY=0` 且 `RELAY_PORT` 正確設定
- 檢查 USB 繼電器是否插入

**車牌辨識不準確**
- 調整攝影機角度和光線
- 確認車牌在畫面中清晰可見
